Livros▼

Digite a referência bíblica em um dos seguintes formatos:
  • Livro capítulo - Mostra o capítulo completo (ex: Mt 5 para mostrar Mateus capítulo 5 inteiro)
  • Livro capítulo,versículo - Mostra um versículo específico (ex: Mt 5,13 para Mateus capítulo 5, versículo 13)
  • Livro capítulo,versículo- - Mostra a partir de um versículo específico (ex: Mt 5,3- para Mateus capítulo 5, versículos 3 em diante)
  • Livro capítulo,início-fim - Mostra um intervalo de versículos (ex: Mt 5,3-12 para Mateus capítulo 5, versículos 3 a 12)
  • Livro capítulo,versículos combinados - Use ponto para separar versículos ou intervalos não consecutivos (ex: Mt 5,3-12.21-38 para Mateus capítulo 5, versículos de 3 a 12, mais os versículos de 21 a 38)
